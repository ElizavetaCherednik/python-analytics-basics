# Python для аналитических задач

**Цель:** Показать базовое владение Python, достаточное для решения учебных аналитических задач.

## Что здесь есть

| Папка | Что демонстрирует |
|-------|-------------------|
| `01_basics/` | Циклы, функции, списки, кортежи, рекурсию, lambda-выражения, генераторы |
| `02_classes/` | Основы ООП: классы, конструктор, методы |

## Используемые технологии
- Python 3.x

## Текущий уровень
- Циклы (`for`, `while`, вложенные циклы)
- Функции (`def`, аргументы, `return`, именованные аргументы)
- Списки и кортежи (срезы, списочные выражения, обход, распаковка)
- Рекурсия (базовое понимание)
- Классы (`__init__`, методы, атрибуты)
- Lambda-выражения
- Генераторы (`yield`)
