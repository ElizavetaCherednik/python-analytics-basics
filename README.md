# Python для аналитических задач

**Цель:** Показать базовое владение Python, достаточное для решения учебных аналитических задач.

## Что здесь есть

| Папка | Что демонстрирует |
|-------|-------------------|
| `01_basics/` | Базовые конструкции: циклы, функции, работу со списками и кортежами, рекурсию |
| `02_classes/` | Основы ООП: классы, конструктор, методы |

## Используемые технологии
- Python 3.x

## Текущий уровень
- Циклы (`for`, `while`, вложенные циклы)
- Функции (`def`, аргументы, `return`, `lambda`, `map`, `filter`)
- Списки и кортежи (срезы, списочные выражения, обход)
- Рекурсия (базовое понимание)
- Классы (`__init__`, методы, атрибуты)
